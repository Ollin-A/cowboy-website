GT America is a commercial typeface from Grilli Type.
A valid web font license must be acquired before production deployment.
Purchase at: https://www.grillitype.com/typeface/gt-america

Place the following files in this directory once licensed:
- GTAmerica-Regular.woff2
- GTAmerica-Medium.woff2
- GTAmerica-Bold.woff2

The @font-face declarations in app/styles/fonts.css are already configured
and will load these files automatically once they are present.
